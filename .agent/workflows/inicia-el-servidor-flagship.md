---
description: Actualiza los cambios en GitHub en español, hace push e inicia el servidor local.
---

// turbo-all
1. Agrega todos los cambios locales.
git add .

2. Realiza un commit con un mensaje automático en español basado en tus cambios.
git commit -m "Actualización automática: Sincronización de cambios antes de iniciar servidor"

3. Sube los cambios a la rama actual en GitHub.
git push

4. Inicia el servidor de desarrollo local.
npm run dev
