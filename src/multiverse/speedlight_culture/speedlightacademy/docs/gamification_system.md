# Gamification System

To drive engagement and viral growth, the platform implements a reward system based on user activity.

## Point Activators
1.  **Participation**: Earning points for every photo "Postulated" to a contest.
2.  **Engagement**: Points for constructive comments on others' work.
3.  **Referrals**: High reward for inviting friends who register and become active users.
4.  **Streaks**: Bonus for daily login or weekly participation.

## Rewards & Levels
*   **Badges**: Visual distinctions on the profile (e.g., "Top Commenter", "Community Builder", "Pro Juror").
*   **Unlockables**:
    *   Access to exclusive contests.
    *   Discounts on courses or merchandise.
    *   **Jury Eligibility**: Points may contribute to the eligibility to run for Juror.
