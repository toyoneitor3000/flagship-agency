# Speedlight Academy

## Overview
Speedlight Academy is the premier educational platform for the automotive culture community, operating under the **Speedlight Culture** ecosystem. 

It resides at [speedlightacademy.com](https://speedlightacademy.com) (pending domain activation) and provides specialized courses in:
- Automotive Photography & Editing
- Mechanics (with Escuela Nacional Automotriz)
- Driving / Drifting / Rally (with Rent for Racing)
- Vinyl Wrapping (with Wrap Masters)

## Integration with Speedlight Culture
This project is designed to work in synchrony with the main [Speedlight Culture](../README.md) platform.
- **Shared Vision**: Empowering car enthusiasts with professional skills.
- **Community**: Students participate in contests and showcase their portfolios on the main Speedlight Culture platform.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
