# PigmentosTkts Design System

## 1. Filosofía Visual
"Vibrant & Dark Modern". La interfaz debe evocar la emoción de un evento en vivo. Fondo oscuro predominante para resaltar los "pigmentos" (colores de acento).

## 2. Paleta de Colores

### Brand Colors
- **Primary (Violeta Eléctrico)**: HSL(262, 83%, 58%) - Acción principal, botones primarios.
- **Background (Deep Night)**: HSL(222.2, 84%, 4.9%) - Fondo principal en modo oscuro.
- **Surface**: HSL(217.2, 32.6%, 17.5%) - Cards, paneles laterales.

### Functional Colors
- **Success**: Verde Neón.
- **Error**: Rojo Intenso.
- **Warning**: Amarillo Ácido.

## 3. Tipografía
- **Font Family**: `Inter` o `Geist Sans` (System default por ahora, optimizado para lectura).
- **Headings**: Bold / ExtraBold. Tracking tight (-0.02em).
- **Body**: Regular. Alta legibilidad.

## 4. Espaciado y Bordes
- **Radius**: `0.5rem` (8px) - Bordes suavemente redondeados, modernos pero no "juguetones".
- **Container**: Centrado con padding `2rem` por defecto.

## 5. Componentes Core (Shadcn Compatible)
Este sistema está diseñado para ser compatible con la arquitectura de componentes de `shadcn/ui`. Las variables CSS en `globals.css` controlan el tema global.
